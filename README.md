# jogjalanjalan-blog
This is the blog app repository.  
https://jogjalanjalan.com/
![jogjalanjalan top image](https://jogjalanjalan.com/wp-content/uploads/2023/05/jogjalanjalan-key-visual.png)

## setup
1. run below command
```
docker-compose up
```

## deploy
1. Simply Static プラグイン > 生成
2. ローカルでの最終確認
```
firebase serve
```
3. デプロイ
```
firebase deploy
```
